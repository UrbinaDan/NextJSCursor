# Daniel Felipe Urbina Sierra - Portfolio

This is a modern, animated portfolio built with Next.js, Tailwind CSS, and Framer Motion.

## Setup

1. Clone the repository or copy the files to your local machine.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) to view your portfolio.

## Features
- Modern, vibrant design
- Responsive and mobile-friendly
- Smooth animations
- Sections: About, Experience, Projects, Education, Contact

Feel free to customize the content and styles!
